The objective of this project was to create a functioning social media website with a set of features such as,
- User Registration,
- Profile Editing,
- Uploading/Editing of Posts,
- Live Chat.

In addition to these features, a set of RESTful API routes were created in order to better help faciliate CRUD operations
being used by the application.

The main development languages in this project were,
- Python - Back end,
- Html - Front end,
- JavaScript - Scripting,
- PostGresql - DB.
